# Պապն ու շաղգամը - The Turnip Game

A browser-based puzzle game based on the Armenian fairytale "The Grandfather and the Turnip". Players must place characters in order to pull turnips from various sources across 10 levels.

## Features

- 10 challenging levels with different turnip sources
- 6 unique characters with special abilities
- Multilingual support (Armenian, Russian, English)
- Introduction story scene
- Progressive difficulty

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:8080`

## Game Mechanics

- Place characters in the correct order to form a chain
- Each character has a strength value and unique ability
- Calculate total strength to pull the turnip
- Different levels require different placement strategies

## Characters

1. Պապ (Grandfather) - Leader, strength: 1
2. Տատ (Grandmother) - Support, strength: 2
3. Թոռ (Grandchild) - Flexible, strength: 1
4. Շուն (Dog) - Dig, strength: 2
5. Փիսո (Cat) - Climb, strength: 1
6. Մուկ (Mouse) - Small, strength: 1
